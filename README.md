# Проект: Место

### Обзор

* Описание
* Используемые технологии

**Описание**

Сервис Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.
* [Ссылка на gh pages](https://temarazin.github.io/mesto/)

**Используемые технологии**

* Семантика HTML5
* БЭМ (nested)
* Flexbox и Grid
* Media queries
* Javascipt
